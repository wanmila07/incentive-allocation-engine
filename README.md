# Informal SME Incentive Optimizer

A constrained multi-objective optimization system for 
SME incentive allocation under competing policy objectives.

## Problem
Malaysia's informal SME incentive programs operate on 
first-come-first-served basis with single-objective 
prioritization. Policymakers must choose between:
- **Efficiency** — high-growth firms
- **Equity** — vulnerable micro-enterprises and rural SMEs  
- **Sustainability** — ESG-aligned, technology-ready firms

No current system handles all three simultaneously under 
real budget constraints.

## Approach
Multi-objective evolutionary optimization (MOEA) to generate 
Pareto-optimal allocation sets — showing policymakers the 
real tradeoffs rather than forcing a single answer.

## Data
World Bank informal enterprise survey data — 5,253 SME records.
